# 🏠 Home Expense

A simple and easy-to-use Home Expense tracker built for Mom.

## Features
- ✅ Add daily expenses
- ✅ Categories (Groceries, Medical, School, etc.)
- ✅ Search and filter
- ✅ Total summary
- ✅ Delete expenses
- ✅ Works on mobile too

## How to Run

1. Install Node.js from https://nodejs.org
2. Clone this repo
3. Run `npm install`
4. Run `npm start`
5. Open http://localhost:3000

## Built With
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Storage: JSON file
